# Blog-Platform-comments

## 📌 Description
This is a simple Blog Platform built using HTML, CSS, and JavaScript. Users can create blog posts, delete posts, and add comments.

---

## 🚀 Features
- Create blog posts
- Delete blog posts
- Add comments to posts
- Dynamic content update
- Beautiful colorful UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📁 Project Files
- index.html
- style.css
- script.js

---

## ▶️ How to Run
1. Download the project folder
2. Keep all files in the same folder
3. Open `index.html` in any browser

---

## 📌 Output
- Blog posts displayed dynamically
- Comments added in real-time
- Posts can be deleted
- Attractive UI design

---

## 👨‍💻 Note
This is a frontend-only project (no database or backend).
